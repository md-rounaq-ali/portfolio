## 🎨 Design & Premium Features

### 💻 1. Interactive Developer Terminal (Linux/Bash Emulator)
A fully functional command-line emulator built directly on the browser using Vanilla JS. It features a complete command buffer allowing recruiters to use their keyboard **Up Arrow (↑)** and **Down Arrow (↓)** keys to navigate through previously typed commands, exactly like a native Bash shell.
* **Supported Commands:** `help`, `about`, `skills`, `projects`, `contact`, `clear`.

### 📋 2. One-Click Copy-to-Clipboard Utility
Recruiters and hiring managers are busy. To minimize friction, the email and phone listings feature a customized hover utility. Clicking copy triggers a hardware-accelerated tooltip feedback loop and changes the icon to a verified green checkmark dynamically.

### ⚙️ 3. Infinitely Scrolling Tech Ticker
An enterprise-grade, GPU-optimized ticker displaying my technical stack. It operates continuously with hardware-accelerated transforms to maintain steady frame rates across ultra-wide monitors and high-refresh screens.

### 🌌 4. Subtle Parallax Project Cards
Each project card utilizes perspective wrapping and dynamic translate filters on scroll, giving a deep sense of luxuriously smooth depth common in high-end tech sites like Apple or Stripe.

### 🤖 5. Invisible JSON-LD Schema (Rich Snippet SEO)
Embedded machine-readable semantic schema structures inside the `<head>` of the page. This tells Google exactly who I am, where I am based, what B.Tech degree I am pursuing, and what technical languages I specialize in, drastically improving organic crawl indexing.

## 💻 Local Installation & Setup

If you wish to run this repository locally on your computer to examine the source code, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/md-rounaq-ali/portfolio.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd portfolio
   ```

3. **Launch the project:**
   * You can open `index.html` directly in any modern browser (Chrome, Edge, Firefox, Safari).
   * **Recommended:** Use the **Live Server** extension in VS Code to see live changes instantly on `http://127.0.0.1:5500`.
